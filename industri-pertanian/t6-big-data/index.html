Big Data

Big Data merupakan variabel data volume besar dengan kecepatan tinggi, kompleks yang membutuhkan teknik dan teknologi tingkat lanjut untuk mengaktifkan pengambilan, penyimpanan, distribusi, manajemen dan analisa informasi.

Karakteristik Big Data

Pada awalnya karakteristik dari big data hanya ada 3V. Namun bertambah seiring berjalannya zaman dan perkembangan teknologi yang begitu pesat. Big data memiliki karakteristik 10V yaitu :

    • Big Data Dalam Pertanian

    Penggunaan big data dalam pertanian dapat sangat membantu untuk meningkatkan hasil pertanian. Petani dapat menggunakan big data untuk mengambil keputusan dengan matang, misalnya terdapat perubahan kelembaban tanah pada bulan tertentu. Jika petani masih menggunakan metode tradisional petani baru akan sadar apabila sesuatu telah terjadi. Namun dengan adanya big data dan penggunaan teknologi, hal ini dapat dicegah sehingga dapat mencegah kegagalan pertanian.
